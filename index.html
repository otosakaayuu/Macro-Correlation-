Pilih event & hasilnya

Pilih event makro yang baru/akan rilis, lalu pilih hasilnya dibanding ekspektasi pasar. Tool akan menampilkan arah probable (LONG/SHORT/NEUTRAL) dan tingkat keyakinan untuk setiap aset yang kamu pantau.

Korelasi antar-aset

Hubungan historis umum antar aset yang sering dipakai trader makro. Korelasi ini statis (basis pola hubungan yang sudah dikenal luas) — gunakan untuk konfirmasi: kalau dua aset berkorelasi kuat tapi bergerak berlawanan dari biasanya, itu sinyal ada divergence yang layak diperhatikan.

Pasangan korelasi terkuat
Diurutkan dari korelasi paling kuat (mutlak)
Korelasi positif (bergerak searah) Korelasi negatif (bergerak berlawanan)
Matrix korelasi
Hijau = positif kuat · Merah = negatif kuat · Abu = lemah/tidak ada data

Kalender macro reference

Daftar event makro rutin yang paling sering menggerakkan crypto, forex, dan komoditas, lengkap dengan jadwal tipikal, apa yang harus diperhatikan, dan kenapa pasar peduli. Tap salah satu hasil untuk lompat langsung ke tabel sinyal.

Semua data & angka probability di tool ini bersifat estimasi heuristik berbasis pola makro umum, bukan hasil backtest statistik dari data live — gunakan sebagai kerangka berpikir, bukan sinyal eksekusi otomatis.
Tool ini berjalan 100% di browser kamu, tanpa API key, tanpa koneksi eksternal, dan tanpa data tersimpan di server manapun.
